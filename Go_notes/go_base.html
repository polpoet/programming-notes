Go Базовые команды

go mod init example.com/hello Создание модуля

В VC Code добавить папку каждого модуля в work space Если открыть все в одной папке ошибка из за того что много go.mod файлов

Go: Install/Update tools Набрать в поиске и установить go tool

go get github.com/xuri/excelize/v2 Скачать с гитхаба модуль

go mod tidy чтобы синхронихировать зависимости модуля

go version посмотреть установленную версию

go build скомпилировать в исполняемый файл

go build -o bin/hello Флаг -o заставляет Go сопоставлять вывод команды с предпочитаемым вами аргументом. В данном случае результатом является новый исполняемый файл с именем hello в подпапке с именем bin.

./hello Запустить исполняемый файл

go mod vendor В Go завендоренные библиотеки должны храниться в папке vendor рядом с файлом go.mod. Чтобы завендорить все используемые библиотеки, нужно выполнить команду go mod vendor.

export CGO_ENABLED=0 Для сборки менять

go env Посмотреть ENV переменные
