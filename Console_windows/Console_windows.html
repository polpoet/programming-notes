Базовые команды

Запуск службы из консоли Если больше двух слов в названии службы, то заключить в кавычки

net start "PostgreSQL 9.6 Server"

net start "postgresql-x64-11"

net start "postgresql-x64-13"

Остановка службы

net stop "PostgreSQL 9.6 Server"
